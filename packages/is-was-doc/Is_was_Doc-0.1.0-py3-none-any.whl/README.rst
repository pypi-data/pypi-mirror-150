Getting Started
+++++++++++++++

#. Setup virtual environment for development

    .. code-block:: shell

        poetry shell

#. Install a dependencies

    .. code-block:: shell

        poetry install

#. List all tasks/targets

    .. code-block:: shell

        nox -l
